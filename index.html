1. Information We Access

To function properly, the Bot temporarily accesses certain information provided by Discord's API during command execution:

  • User IDs: Discord user identification numbers (accessed temporarily, not stored except for premium users)
  • Server IDs: Discord server (guild) identification numbers (accessed temporarily, not stored)
  • Message content: Only when necessary to execute specific commands
  • Profile pictures: Processed in real-time for modification features, then immediately discarded

Important: Except for premium user IDs, all of this information is processed in memory and is never written to disk, databases, or any permanent storage. Once a command completes, all associated data is immediately discarded.

2. How We Process Information

The Bot operates primarily in real-time:

  • Commands are processed instantly in memory
  • Profile picture modifications are performed temporarily and the results are sent back to Discord
  • Moderation actions are executed immediately without logging
  • No data persists after command execution completes (except premium user IDs)

3. Data Retention

General Users: We retain absolutely no data. There are no logs, no databases, and no files created or stored. Every interaction with the Bot is ephemeral and exists only during command execution.

Premium Users: We store only your Discord User ID to maintain your premium status across bot restarts and server maintenance. This ensures uninterrupted access to premium features. No other personal information, usage data, or activity logs are stored.
